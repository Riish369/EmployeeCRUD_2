# Employee Management System (Java + PostgreSQL)

## Features
- Add Employee
- View Employees
- Update Employee Name
- Delete Employee

## Tech Used
- Java (JDBC)
- PostgreSQL

## Project Structure
- db → Database connection
- dao → Database operations
- model → Data structure
- Main → CLI interface

## How to Run
1. Start PostgreSQL
2. Create database: tutdb
3. Update DB credentials in DBConnection.java
4. Run Main.java
